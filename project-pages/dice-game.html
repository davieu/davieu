Dice Game

Live Demo

Formally known as Pig. A simple web-app implementing game logic and buttons that interact and mutate data

Just a fun dice game that I coded better known as pig. Coding this web-app helped me learn some good CSS tricks to keep the UI looking clean and interactive. I had loads of fun implementing type-coercion with the dice pictures to cycle through them on each roll and adding and removing classes to give the UI a winner. It is deployed on Heroku but it is strictly frontend.
